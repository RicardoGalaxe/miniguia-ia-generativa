# 📘 Miniguia de Estudos: Inteligência Artificial Generativa

## 🎯 Contexto e Objetivos
Este projeto foi desenvolvido como parte do desafio da DIO para criar um **Caderno Temático no NotebookLM**, utilizando técnicas de curadoria de fontes e engenharia de prompts.

**Tema:** Fundamentos de Inteligência Artificial Generativa

**Objetivos:**
- Compreender os principais conceitos e arquiteturas de IA generativa
- Identificar aplicações práticas no mercado
- Desenvolver habilidades para extrair o máximo de ferramentas de IA

## 📖 Glossário

| Termo | Definição |
|-------|-----------|
| **LLM** | Large Language Model - modelo treinado em bilhões de parâmetros de texto |
| **Fine-tuning** | Ajuste fino de um modelo para uma tarefa específica |
| **Alucinação** | Quando o modelo gera informações incorretas com alta confiança |
| **Transformer** | Arquitetura que usa mecanismo de atenção para processar sequências |
| **Multimodal** | Capacidade de processar texto, imagem, áudio e vídeo |
